# GitHub 完全指南：挖掘 GitHub 的全部潜力

> 一份面向中文用户的 GitHub 知识课程，系统性地覆盖 GitHub 每一项功能，从基础操作到高级生态，助你完全挖掘 GitHub 的使用潜力。

## 课程定位

本仓库解决一个核心问题：**大多数人只用了 GitHub 10% 的能力。**

GitHub 不仅仅是代码托管平台——它是协作引擎、自动化平台、知识管理系统、AI 开发环境、项目治理工具，以及开源生态的基础设施。本课程逐一拆解每个功能模块，让你从"会用 Git"进化到"精通 GitHub"。

## 目标读者

- 已有基本 Git 操作经验的开发者
- 想系统掌握 GitHub 全部功能的团队负责人
- 希望建设开源项目的维护者
- 对 GitHub Actions / Copilot / API 集成感兴趣的技术人员
